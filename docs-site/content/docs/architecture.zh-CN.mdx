# 架构

SkillsGo 是一个包含三个产品上下文和一个公共文档工作区的 monorepo。

| 上下文 | 职责 |
| --- | --- |
| 桌面应用 | 用户体验、导航、发现和安装流程 |
| CLI | 本地文件系统变更、安装计划、清单、锁和 Agent 适配器 |
| Hub | 公共 Skill 身份、不可变制品、元数据、搜索、排序和安装事件 |
| 文档站点 | 面向用户的产品与开发者文档 |

## 集成方向

App 不会直接调用 Hub。它通过稳定的机器可读契约调用内置 CLI。CLI 从 Hub 解析公共信息和制品，然后将托管内容投射到各 Agent 目标。

```text
App --JSON/NDJSON--> CLI --HTTP--> Hub
                       |
                       +---------> Store 与 Agent 目标
```

这条边界让本地变更策略集中在一个引擎中，避免图形界面与终端产生不兼容的安装行为。
