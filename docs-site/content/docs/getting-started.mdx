# Getting Started

SkillsGo lets you discover public Skills, inspect their metadata, and install them into supported Agent targets.

## Before you begin

Choose the interface that matches your workflow:

- Use the **desktop App** for visual discovery and installation journeys.
- Use the **CLI** for terminal workflows and automation.

The production desktop App bundles a compatible CLI, so App users do not need to install the CLI separately.

## Typical journey

1. Search the Hub for a Skill.
2. Review its public identity, version, and source metadata.
3. Select the user-level or workspace-level Agent targets.
4. Install the immutable artifact.
5. Update or remove the managed installation through SkillsGo.
