# Require First-launch Onboarding

SkillsGo gates clean installations on a two-step, resumable Mandatory Onboarding instead of opening Discover immediately. The interruption is justified by the need to establish the product as the user's Agent Skills Store, show the locally detected Agent boundary, and obtain an explicit project choice before local management begins; the journey remains short, permits permanent project skipping, starts no speculative background scan, and does not run for existing users after upgrade.
