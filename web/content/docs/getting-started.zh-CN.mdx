# 快速开始

SkillsGo 帮助你发现公共 Skills、查看其元数据，并将它们安装到受支持的 Agent 目标。

## 开始之前

选择适合你工作方式的界面：

- 使用**桌面应用**完成可视化发现与安装流程。
- 使用 **CLI** 完成终端操作和自动化。

正式版桌面应用会内置兼容的 CLI，因此桌面应用用户不需要单独安装 CLI。

## 典型流程

1. 在 Hub 中搜索 Skill。
2. 查看它的公共身份、版本和来源元数据。
3. 选择用户级或工作区级 Agent 目标。
4. 安装不可变制品。
5. 通过 SkillsGo 更新或移除托管安装。
