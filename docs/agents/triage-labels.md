# Triage Labels

The engineering skills use five canonical triage roles. This table maps each role to the label used in this repository's issue tracker.

| Canonical role | Tracker label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | A maintainer needs to evaluate the issue |
| `needs-info` | `needs-info` | The issue is waiting for information from the reporter |
| `ready-for-agent` | `ready-for-agent` | The issue is fully specified and ready for an autonomous agent |
| `ready-for-human` | `ready-for-human` | The issue requires human implementation or judgment |
| `wontfix` | `wontfix` | The issue will not be actioned |

When a skill refers to a canonical role, use the corresponding tracker label from this table.
