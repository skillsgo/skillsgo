# Name the Public Distribution Context SkillsGo Hub

SkillsGo names its public Skill discovery and distribution product, domain context, workspace, executable, configuration namespace, and client vocabulary **SkillsGo Hub**. “Registry” was technically accurate but sat at a different abstraction level from SkillsGo App and SkillsGo Cloud; Hub provides one product name for Skill identity, search, immutable artifacts, trust metadata, and distribution while preserving existing Skill coordinates and sums. SkillsGo Cloud independently owns installation events and ranking projections.
